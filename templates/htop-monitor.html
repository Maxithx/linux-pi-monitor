Below is the Glances real-time system monitor. Glances provides live stats for CPU, RAM, network, disks, and more.

Glances key bindings (click inside the monitor first to focus)
Views & layout
  • 1 Toggle per-CPU bars
  • 2 Toggle left sidebar
  • 3/4/5 Cycle CPU/mem graphs detail
  • d Toggle Disk I/O, f Filesystems, n Network, s Sensors
  • D Docker stats (if available)
  • h / ? Toggle built-in help
Process sorting
  • a Auto (by most used resource)
  • c By CPU%, m By MEM%
  • p By process name, u By user
  • i By I/O rate, t By TIME+
Alerts & misc
  • l Toggle alert log
  • w Clear warnings, x Clear all alerts
  • q / Esc Quit (for TUI; close tab for Web UI)
Tip: Keyboard shortcuts also work in the Web UI—just make sure the black monitor area has focus (click it once). For the full list, press h inside Glances.
